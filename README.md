# Upgradable Item
A plugin let Tools or Sword have a upgrade system
# Usage:
```
<> is required
[] is optional

/upi get <Tools> [rarity]
```
# Available Tools:
```
Sword
Axe
```
# Rarity request:
```
Wooden  ==> Stone:     10XP
Stone   ==> Golden:    50XP
Golden  ==> Iron:      80XP
Iron    ==> Diamond:   5000XP
Diamond ==> Netherite: 50000XP
```
# How to build
Request:
  - Java Jdk 17
  - Maven
```
maven package
```
